Support tool for fRAme: An Abilities-Driven Framework for Evaluating Human Augmentation or Replacement through Autonomous Systems.

This web page provides a tool for evaluating autonomous systems' abilities, with a specific focus on those ones considered relevant for replacement or augmentation.

You can download an example of the application of the tool used for the evaluation of Alexa here.
